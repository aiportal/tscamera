Getting Started

Terminal Camera is a comprehensive and practical software utility that provides you with a simple means of monitoring computer usage and user activity. With the help of Terminal Camera you have the possibility to track each opened application, playback all user activity within each section. It also provides you with an embedded video player so you can preview the selected action.

Install

Installation of this product requires system administrator privileges.

  1. Unzip compressed files you have downloaded.
  2. Run TSCamera.exe to install.
  3. If .NET 2.0 Framework not installed, it will be download and install automatically.

Login

Visit http://localhost:1088 or http://127.0.0.1:1088 by web browser.

Default account/password is admin / admin.

Buy And Register

  2. Click Buy Now button to buy this software and MachineId will be send automatically.

  4. Copy and paste license key in register box and click Register button to register.
  5. After the registration done, right panel of Buy Now button and adware image will be invisibly.

Management

User can administrate Terminal Camera by web browser from localhost, LAN or Internet.

Local Computer

Visit http://localhost:1088 or http://127.0.0.1:1088 by web browser.

LAN

  1. Run cmd.exe and typed "ipconfig" command at local computer.
  2. Get IP address from response.
  3. Typed http://ip:1088 (Example: http://192.168.56.101:1088) into web browser address bar at any computer in LAN.
  4. If can't access, please check firewall rules.

Internet

If you're connected to internet directly, you can access management interface like LAN. if you're behind a router, make a port mapping from your computer to internet.

Configuration

System Parameters

    System Parameters

  • Monitor Enabled: Enable or disable monitoring.
  • TrayIcon Visible: Show or hide icon at notification bar.
  • GrayScale Image: Store grayscale image for fewer disk space or colorful image for clear movie.

    • Access Rule

    You can limit management interface access to user or computer by this parameters.

  • Forbid Local Users: Only one user who installed Terminal Camera can visit http://localhost:1088.
  • Permit Remote Address: Only permited computer can visit management interface by http://ip:1088, and this parameter supprot ipv6 .
    The example is: http://192.168.0.99:1088

    • Data Limit

    You can limit data storage space by this parameters.

  • Disk Quote: The disk quota for movie files. if overflow, delete old files automatically.
  • Days Persist: The days number to keep movie file on disk. if expired, delete old files automatically.

    • LDAP Configuration

    You can filter snapshots by Domain user or group.

  • Domain Name: Domain name for LDAP query.
  • Manager Name: LDAP query account.
  • Manager Password: LDAP query password.
  • Organizational Unit: Optional parameter for LDAP Organizational Unit.

Filter Policy

  • Users: Select users to monitor or not monitor.

    • If LDAP Configuration parameters has been set, this panel will list domain users.

  • Groups: Select user groups to monitor or not monitor.

    • If LDAP Configuration parameters has been set, this panel will list domain groups.

  • Program: Select programs to monitor or not monitor.
